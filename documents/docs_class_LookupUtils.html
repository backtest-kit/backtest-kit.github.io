LookupUtils

In-memory registry of currently running backtest and live activities.

Purpose:

  • Each Backtest.run / Live.run / per-strategy walker iteration registers an IActivityEntry on start and removes it on completion.
  • Candle.spinLock consults isParallel to decide whether the event-loop hand-off (post-candle-fetch spin) is worth performing. With a single active workload there is no peer to yield to, so the spin is skipped entirely.

Exposed as the Lookup singleton; no constructor parameters.

removeActivity

removeActivity: (activity: IActivityEntry) => void

Removes a previously registered activity from the lookup map. Must be paired with a prior addActivity, typically in a finally block, so a thrown error in the underlying run does not leave a stale entry behind.
