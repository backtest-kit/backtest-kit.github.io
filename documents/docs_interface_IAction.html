IAction

Action interface for state manager integration.

Provides methods to handle all events emitted by connection services. Each method corresponds to a specific event type emitted via .next() calls.

Use this interface to implement custom state management logic:

  • Redux/Zustand action dispatchers
  • Event logging systems
  • Real-time monitoring dashboards
  • Analytics and metrics collection
